Install

The Ecsact SDK contains all the necessary tools for developing with Ecsact. All integrations requires the Ecsact SDK.

Ecsact SDK

Ecsact SDK on Windows 10+ is distributed as an app installer and is the recommended way to install the Ecsact SDK. Other download options are available on our GitHub releases page.

download ecsact_sdk.appinstaller

An archive (tar.gz) of the latest Ecsact SDK release can be found on our GitHub releases page for Linux.

info Ecsact SDK is not available on macOS (yet.) To keep up to date or provide feedback please follow the macOS release tracking issue on GitHub.
