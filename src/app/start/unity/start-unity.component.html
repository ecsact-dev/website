Setting Up Unity

Ecsact Integration Package

Unity Ecsact Integration is provided as a Unity Package and is required to use Ecsact with Unity. Installing the latest integration is easy.

  1. Start your unity project
  2. Open the Package Manager window ( Window arrow_right Package Manager)
  3. Click the add_box in the top left
  4. Add the Ecsact Unity integration git URL 
    						git@github.com:ecsact-dev/ecsact_unity.git
NOTE: You may get errors about missing components that you will need to install. Ecsact uses vswhere to output these errors, and you can search the outputs for entries

Assembly

For systems to be registered into Ecsact, it needs to be part of an Assembly. In addition, it will need the following dependencies:

  1. Ecsact
  2. EcsactCSharpSystemImplRuntime

After you've setup your assembly it needs to be set in Edit arrow_right ProjectSettings arrow_right Ecsact under System Implementations

Unity Sync

Enabling Unity Sync is very simple. All you have to do is go to Edit arrow_right ProjectSettings arrow_right Ecsact and checkmark Enable Unity Sync. When enabled, it will search for any Unity Sync files automatically and add them to its pool which is listed below.
