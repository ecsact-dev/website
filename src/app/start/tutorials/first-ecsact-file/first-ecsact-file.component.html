Tutorial: Your first Ecsact file

Introduction

In this tutorial, you will create your first Ecsact file. Later in other tutorials this Ecsact file will be the basis for learning how to write systems implementations.

An Ecsact file can be considered a written contract for how your simulation works. State is declared with components and execution is declared with systems and their system capabilities.

The Package Statement

Every Ecsact file starts with a package statement.


				package first.project;

The package name is similar to a namespace in other languages except it must be unique. Later on when you have multiple Ecsact files you can also import packages with this name.

First Component

Next up we declare the state for our simulation. This is done with components. Let's start off with simple common component called Position. 


				component Position {{'{'}}
					f32 x;
					f32 y;
				{{'}'}}

We'll use this Position component to represent our entities coordinates in the world. Before adding more state let's move onto adding a system that will use this component.

First System

Now that we have some state in the form of our Position component, lets do something with it! System declarations specify system capabilities which states what components a system is allowed to do with which components. 


				system Gravity {{'{'}}
					readwrite Position;
				{{'}'}}

There we go! We've specified that we want to readwrite the component Position. Our system Gravity will run on only entities that have the component Position and on no others.

System declarations are only 1 part of functional system. The other part is the system implementation. This tutorial doesn't go over system implementations. In the next steps you'll have the option to learn about implementing systems in various languages.

Full Ecsact File

This is the full Ecsact file from this tutorial for your convenience!


				package first.project;

				component Position {{'{'}}
					f32 x;
					f32 y;
				{{'}'}}

				system Gravity {{'{'}}
					readwrite Position;
				{{'}'}}
