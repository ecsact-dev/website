Setting Up Unreal

Plugin Installation

To get started using Ecsact with your Unreal project we highly recommend installing the Ecsact Unreal plugin. At the time of writing this the Ecsact plugin has not been published to Fab (tracking issue on github.) For now the release archive are available on the Ecsact Unreal plugin Github releases page. Extract EcsactUnreal-Win64.zip to your Unreal engine marketplace plugins directory to install. 


				# Ecsact Unreal zip contents should be extracted here
				[UE-Install-Dir]/Engine/Plugins/Marketplace/Ecsact

Once you have installed the Ecsact Unreal plugin go to Edit arrow_right Plugins. Find the Ecsact plugin and make sure it's enabled for your project. You may have to restart your editor after enabling the plugin.

Ecsact Plugin Settings

After enabling the Ecsact plugin and restarting your editor you should have new project settings specifically for Ecsact. You can find the settings in Project Settings arrow_right Plugins arrow_right Ecsact.

Here you are able to configure and enable/disable different features of the plugin. For your first setup we recommend enabling the build system and adding the rt_entt build recipe.

Whats next?

From this point on when you have the editor open the Escact Runtime will be built automatically when changes to .ecsact in your Source directory change. If no Ecsact files are detected you should get a warning in your editor output log. If this is your first Ecsact project then we recommend reading the your first Ecsact file tutorial. Otherwise, take a look at some of the Ecsact Unreal plugin specific features below for further reading.
