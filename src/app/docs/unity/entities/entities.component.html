Creating Entities in Unity

There are multiple ways to create entities in Unity with Ecsact

Using executionOptions

The Runner can be used to create entities through its options. It has an optional callback with the new entity's ID 

Ecsact.Defaults.Runner.executionOptions.CreateEntity((entityId) ={{'>'}} {{'{'}}{{'}'}});

CreateEntity also returns a builder that allows it to be created with initial components 


					var fooComponent = new example.Foo {{'{'}}.someValue = 5{{'}'}};
					var barComponent = new example.Bar {{'{'}}.someOtherValue = 10{{'}'}};

					Ecsact.Defaults.Runner.executionOptions
						.CreateEntity((entityId) ={{'>'}} {{'{'}}{{'}'}})
						.AddComponent(fooComponent)
						.AddComponent(barComponent);

Dynamic Entity

A convenient Monobehaviour that allows you to create an entity and add components to it through the Unity UI

Note: The dynamic entity uses SetPreferredEntityGameObject when Unity Sync is enabled.

Generating Entities

Entities can be generated within Ecsact by declaring it and calling it through the context with .Generate
