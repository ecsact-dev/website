Wasm Systems Overview

Wasm (or Web Assembly) is a portable binary format perfectly suitable for executing code you write on the web or embedded in another program. Ecsact supports having your system implementations in the Wasm format. System implementations compiled to Wasm are the best all around way for a working with Ecsact.

This article should give you everything you need to know in order to create a valid system implementation .wasm file.
